Math Service Web Application

Welcome to the Math Service API built with our custom web framework!

Available REST Endpoints:

Hello Service

GET /App/hello?name={yourName}

Returns a personalized greeting.

Try it: /App/hello?name=Pedro

PI Constant

GET /App/pi

Returns the value of PI (π).

Try it: /App/pi

Euler's Number

GET /App/euler

Returns Euler's number (e).

Try it: /App/euler

Square Calculation

GET /App/square?num={number}

Calculates the square of a number.

Try it: /App/square?num=5

Sine Calculation

GET /App/sin?angle={radians}

Calculates sine of an angle in radians.

Try it: /App/sin?angle=1.5708 (π/2)

Cosine Calculation

GET /App/cos?angle={radians}

Calculates cosine of an angle in radians.

Try it: /App/cos?angle=0
